## 📦 Installation

Clone repository:

```bash
git clone https://github.com/your-username/litewallet.git
cd litewallet
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🚀 Usage

1. Buat wallet baru atau import wallet yang sudah ada
2. Simpan private key / seed phrase dengan aman
3. Mulai kirim, terima, dan kelola aset crypto
4. Hubungkan ke aplikasi Web3 favorit Anda

---

## 🔐 Security

Litewallet mengutamakan keamanan:

* Non-custodial (Anda memegang kendali penuh atas aset Anda)
* Private key tidak pernah disimpan di server
* Proteksi terhadap serangan umum di Web3
